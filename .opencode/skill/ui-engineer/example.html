Modern Business Process Management

Kees is a comprehensive BPM platform built with modern technologies, helping teams manage work items, projects, cases, and processes efficiently. Built with Svelte 5, Tailwind CSS, and powered by PostgreSQL.

Core Features

Everything you need to manage your business processes

Work Management

Work Items

Unified task system with process, manual, and help task types. Kanban board with drag-and-drop, priority levels, deadline tracking, and multi-assignee support.

Projects

Project Management

Project CRUD operations, member management, public sharing via tokens, project-level permissions, and comprehensive statistics.

Cases

Case Management

Business case tracking with workflow steps, task assignment, process integration, and multi-step case workflows.

Process

Process Management

Business process definitions, task templates, step management, process-based task generation, and case-process integration.

Chat

Chat & Messaging

Team chat for internal communication, AI assistant mode, message threading, chat history, and real-time messaging.

Analytics

Analytics & Reporting

Client-side analytics, user behavior tracking, employee reports, operational reports, dashboard statistics, and hours tracking.

Technology Stack

Built with modern, scalable technologies

Framework Svelte 5

Architecture Highlights

Modern architecture for scalability and maintainability

Component-Based UI

108+ reusable Svelte components following the gray-html design system for consistent UI across the application.

Layered Architecture

Clear separation of concerns with routes, services, stores, utilities, and API layers for maintainability.

Type Safety

End-to-end TypeScript with Zod validation for runtime type safety and better developer experience.

Optimistic Updates

Fast UI with optimistic updates and background sync for real-time collaboration feel.

RBAC

Role-based access control with granular permissions for users, projects, and resources.

6 Themes

Multiple themes including default, catalyst, pglite, planet, trevor, and cursor for personalization.
