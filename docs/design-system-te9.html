// principles

Design Principles

TE9-P01

Terminal-native aesthetic

The interface speaks the language of terminals. Monospaced body text, prompt-style branding, lowercase labels. It feels like home for developers who live in the command line.

TE9-P02

Dark by default

The dark palette isn't a theme toggle — it's the foundation. #0a0a0a backgrounds with carefully separated surface layers. Every shade has a role. No guesswork.

TE9-P03

Dual accent harmony

Amber (#ff9500) for action and emphasis. Emerald (#10b981) for status and selection. Two accents, distinct roles. Never compete, always complement.

TE9-P04

Progressive reveal

Staggered entrances, hover-triggered gradients, subtle glow effects. Information unfolds. Nothing shouts. The interface respects attention by revealing on demand.
