What's happening this cycle

Three currents run through this batch. First, AI agents are acquiring hands: Browser Harness gives LLMs direct control of a live browser via Chrome DevTools Protocol, while WriteHERE decomposes writing tasks into recursive sub-plans that adapt on the fly. The agent wave isn't coming — it's lapping at the shore.

Second, the edge is real. PicoLM runs a billion-parameter model on a $10 board with 256 MB RAM. That's not a toy — it's a signal that offline-first, privacy-preserving AI is becoming a practical deployment target, not just a research curiosity.

Third, developer infrastructure keeps maturing. useSend gives you a self-hosted alternative to Resend and SendGrid. agx wraps analytics into a Tauri+SvelteKit desktop app. Forg turns git commits into social posts. And TRIP proves you can self-host your own trip planner without surrendering data to a SaaS. The common thread: builders taking back control of their toolchain.

The latest agent tooling isn't about writing code faster — it's about letting AI perceive and act on the real web, compose long-form content, and run on hardware you'd find in a junk drawer.

browser automation tool

Browser Harness — LLMs Get a Live Browser

At roughly 592 lines of Python, Browser Harness connects an AI coding assistant (Claude Code, Codex) directly to Chrome via the DevTools Protocol. The twist: the agent writes and edits its own helpers.py mid-task, creating a self-healing automation loop that adapts when selectors break or UIs change. No brittle scripts. No replay maintenance.
When to use it
Complex form submissions, multi-step browser testing, scraping dynamic sites — anywhere traditional Playwright or Selenium scripts would rot within weeks.

WriteHERE — Recursive Planning for Long-Form Writing

github.com
Most LLM writing tools follow a rigid prompt–generate pipeline. WriteHERE takes a different approach: it recursively decomposes a writing task into sub-tasks, plans each one, and integrates retrieval, reasoning, and composition at every level. The result is adaptive long-form output — technical reports, creative fiction — that isn't shackled to a fixed template.
When to use it
When you need multi-page technical docs, grant proposals, or any writing task where rigid outlines fail and the AI needs to re-plan as it goes.
LLM integration recursive planning long-form writing research
github.com/principia-ai/writehere ›
ai/ml library

PicoLM — 1B Parameters, $10 Hardware

github.com
A from-scratch C11 inference engine that runs a billion-parameter model on hardware with as little as 256 MB RAM. By memory-mapping model files, PicoLM keeps runtime memory around 45 MB. Pair it with PicoClaw for fully offline AI agents on Telegram, Discord, or CLI.
Why it matters
This is edge AI's proof point. When a Raspberry Pi runs a billion-parameter model, the economics of privacy-first and offline-first apps change completely.

Own Your Toolchain

Self-hosted email, desktop analytics, and a platform that turns commits into credibility. The through-line: taking back control of the infrastructure you depend on.

email infrastructure

useSend — Self-Hosted Transactional Email

github.com
An open-source alternative to Resend, SendGrid, and Postmark. useSend gives you a REST API for transactional and marketing emails, domain management, webhook integration, scheduled sends, and a deliverability dashboard — all self-hostable. No vendor lock-in, no per-email pricing surprises.
When to use it
When your team needs full control over email delivery, custom analytics, or integration with internal systems that SaaS providers can't reach.
email open source self-hosted SMTP API
github.com/usesend/usesend ›
desktop application framework

agx — AI-Powered Analytics Desktop App

github.com
Built with Tauri + SvelteKit + Plot, agx is a cross-platform analytics tool with an interactive SQL editor, schema browser, and LLM integration via Ollama. Run it native or web-based via Docker Compose. It targets data teams who want a local-first analytics interface with AI assistance baked in.
When to use it
When you need a custom analytics dashboard with ClickHouse UDFs, local LLM query assistance, or a desktop-first experience that doesn't require a cloud deployment.
Rust SvelteKit Tauri analytics SQL editor

Forg — Ship, Share, Sell

forg.to
Forg aggregates your tech stack, GitHub activity, and commit history into a verified builder profile. Its Creator Studio then generates social posts from raw code commits and schedules them to X, LinkedIn, and Bluesky. It's a marketing engine for developers who'd rather write code than craft tweets.
When to use it
When you need to build professional credibility, automate dev-content distribution, or aggregate your open-source identity into a single verifiable profile.

Atomic — Connected Knowledge Base

atomicapp.ai
Atomic converts unstructured notes into a semantic knowledge graph with vector-embedding search. It integrates with MCP clients like Cursor, so you can query your entire note library semantically while coding. Features include agentic chat, wiki synthesis from raw atoms, and force-directed graph visualization.
When to use it
During research and documentation phases, when you need to surface related concepts across hundreds of notes or synthesize complex topics into structured wikis.

SuperSplat — Browser-Based 3D Gaussian Splat Editor

github.com
Built on PlayCanvas, SuperSplat is a free, open-source tool that lets you inspect, edit, optimize, and publish 3D Gaussian splats entirely in the browser. No local install, no heavy software stack. Load your splats, clean them up, and export — all through a web interface running at localhost:3000 after a quick npm run develop.
When to use it
When you need immediate, lightweight editing of complex 3D spatial data without installing desktop software — prototyping, web-based AR/VR, or spatial visualization workflows.
3D graphics Gaussian splatting PlayCanvas web editor

HappyHorse — AI Video with Native Lip-Sync

happyhorse.mov
HappyHorse generates character-driven short films with multi-shot consistency, 1080p output, and native lip-sync across multiple languages. The four-step workflow (script → render → refine → export) makes it viable for commercial campaigns, motion graphics, and rapid video prototyping without a production crew.
When to use it
Rapid prototyping of character-driven content, localized ad campaigns with lip-sync, or any situation where traditional video production is too slow or expensive.

TRIP — Self-Hosted Trip Planner with POI Maps

github.com
A minimalist, Docker-deployable web app for tracking points of interest on interactive maps and planning multi-day itineraries. TRIP supports OIDC authentication and keeps all your travel data on your own server. It's also a solid reference project for anyone building mapping applications with collaborative features.
When to use it
When you want a privacy-focused trip planner without third-party services, or as a learning project for full-stack mapping apps with Docker deployment.
