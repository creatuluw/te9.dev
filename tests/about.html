About te9.dev

te9.dev is an AI-powered development workspace that implements the te9-method for structured, testable, and traceable software development with AI agents. We combine AI agents with structured workflows to deliver high-quality, well-documented software with complete traceability.

Core Philosophy

  • ✓ Structure First: All development work follows PRD-driven workflows
  • ✓ Memory-Enabled: Every interaction is tracked and contextualized through OpenMemory
  • ✓ Test-Driven: Every PRD must pass acceptance criteria and test suites
  • ✓ Design-System-Driven: Consistent UI through the Gray Design System
  • ✓ Traceable: Complete audit trail of all decisions, implementations, and progress

PRD-Driven Development

Create, manage, and execute Product Requirements Documents with structured interviews, automatic file generation, and complete audit trails.

OpenMemory Integration

Maintain persistent context across 5 memory sectors: episodic, semantic, procedural, emotional, and reflective interactions.

Multiple UI Frameworks

Generate components with Gray Design System, UIKit, DaisyUI, and Melt UI - 160+ components across 4 frameworks.

Browser Automation

Automated testing and web scraping with Playwright and Chrome DevTools for performance analysis and interaction testing.
