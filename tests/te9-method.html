Philosophy

Core Principles

Workflow

Mandatory Workflows

Two critical workflows that must be followed for all development activities.

Workflow 1: Memory Integration - QUERY → ANALYZE → INCORPORATE → STORE
  • QUERY FIRST

    Query OpenMemory before responding to any request

    openmemory_query
  • ANALYZE

    Review results and identify patterns, relationships, and preferences

  • INCORPORATE

    Adapt response based on memory context and user preferences

  • STORE LAST

    Save important learnings after every interaction

    openmemory_store
Workflow 2: PRD-Driven Development - INTERVIEW → PLAN → CREATE → EXECUTE → TEST → TRACK
  • INTERVIEW

    Conduct structured 9-question requirements gathering

    prd-interview
  • PLAN (Optional)

    Split large work into multiple well-sized PRDs with dependencies

    prd-plan
  • CREATE

    Generate PRD files, configuration, and execution prompts

    prd-create
  • EXECUTE

    Implement work according to acceptance criteria

    prd-execute
  • TEST

    Verify all criteria pass, no regressions introduced

    prd-test
  • TRACK

    Log progress and maintain complete history

    prd-track
